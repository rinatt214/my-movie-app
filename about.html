🎬 关于本项目

这是一个基于 TMDB API 的电影推荐系统，本项目作为完整前端练习工程构建，已上线部署。

🛠 当前功能

  • 首次使用时，从 TMDB 实时加载热门电影数据
  • 逐部展示 10 部电影，用户选择“喜欢 / 不喜欢 / 没看过”
  • 基于喜欢的类型智能推荐新电影，支持评分优先
  • 显示电影封面、评分、简介、类型等卡片式信息
  • 收藏功能 + 收藏图表分析（类型偏好、评分分布）
  • 响应式设计支持手机访问
  • 深色模式切换（开发中）
  • 项目采用模块化结构（HTML / CSS / JS 多文件）

📂 项目结构

  • index.html：主页面
  • style.css：样式文件
  • js/ 文件夹中包含：
    • api.js：数据获取（TMDB）
    • logic.js：用户交互逻辑
    • ui.js：页面渲染（卡片、推荐、图表）
    • utils.js：辅助函数
    • main.js：页面入口与初始化

🔮 后续计划

  • 增加类型筛选器（只看“剧情+动作”等）
  • 支持多语言切换
  • 分享推荐结果
  • 支持多用户切换记录
