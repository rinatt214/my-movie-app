🎬 关于这个项目

这是一个基于用户偏好的电影推荐系统，作为我学习前端开发 + 产品设计的练手项目。

💡 项目功能（当前已实现）

  • 首次打开页面，逐一展示10部随机电影
  • 用户对每部电影选择“喜欢 / 不喜欢 / 没看过”
  • 根据“喜欢”的类型推荐电影
  • 支持继续推荐（不限一轮）
  • 收藏电影、查看收藏列表
  • 数据保存在本地浏览器，页面刷新不丢失

📌 使用方式

  1. 点击 首页 进入推荐页面
  2. 对每一部电影作出反馈
  3. 完成10部后获取推荐结果
  4. 可以收藏 / 再推荐

🛠️ 后续计划升级

  • ✨ 推荐理由解释（“因为你喜欢剧情片”）
  • 📊 收藏或喜好统计图表（Chart.js）
  • 🌐 支持国家 / 语言 / 类型筛选器
  • 📱 页面响应式优化（适配手机端）
  • 🧩 支持导入用户自定义电影数据
  • 🔗 数据源改为在线获取（TMDB API）

📦 数据来源说明（计划）

当前为自定义静态电影数据（20部）
计划后期接入 TMDB API 实现动态获取真实数据，包括评分、类型、封面图、语言等。
