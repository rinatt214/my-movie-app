🎬 关于本项目

这是一个基于 TMDB API 的电影推荐系统，作为完整前端练习项目构建并已上线部署。

🛠 当前功能

  • 🆕 首次使用时，从 TMDB 实时加载热门电影数据
  • 📽️ 逐部展示 10 部电影，用户选择 👍 喜欢 / 👎 不喜欢 / ❓ 没看过
  • 🎯 基于偏好类型持续推荐，含 2:1 探索推荐策略
  • 🖼️ 显示封面、评分、简介、类型等卡片式信息
  • ⭐ 收藏功能 + 图表分析（类型偏好、评分分布）
  • 📱 响应式设计支持手机访问
  • 🌙 深色模式切换（开发中）
  • 🧩 项目采用模块化结构（HTML / CSS / JS 分离）

🧠 推荐逻辑说明

系统记录你对每部电影的反馈。当你标记“喜欢”的电影属于某些类型（如 🎞️ 剧情 / 🤖 科幻），系统会优先推荐相同类型中你还没看过的电影。 同时也会以 2:1 的比例偶尔推荐其他类型电影，以鼓励探索兴趣边界 🎲。推荐逻辑基于 userChoices 和电影类型动态更新。

📂 项目结构

  • index.html：主页面
  • style.css：样式文件
  • js/ 文件夹中包含：
    • api.js：从 TMDB 获取数据
    • logic.js：处理用户选择与状态
    • ui.js：渲染推荐卡片与收藏功能
    • utils.js：辅助函数，如评分格式化
    • main.js：初始化与控制流程入口
